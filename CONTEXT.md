# Context Glossary

## Character Guessing

- **Character**: A Detective Conan character that players try to identify.
- **Clue**: A text-only property or trait that describes a character. Images are out of scope for questions.
- **Round**: One character guessing prompt that shows all five text clues at the start of the round.
- **Match**: A realtime room game made of ten rounds.
- **Round timer**: Each round gives players twenty seconds to choose an answer.
- **Reveal phase**: After each round ends, players see the correct answer, their own answer, and the latest ranking for five seconds.
- **Answer roster**: The full list of fifty available character names shown to players as searchable answer choices for every round.
- **Seed character data**: The initial game ships with fifty Thai-language Detective Conan characters, each with five text clues that can be edited later.
- **Visual style**: The game uses a dark detective-board interface with a prominent timer and clue board.

## Online Competition

- **Realtime room**: A shared match space for up to eight players where players join with a room code, answer the same rounds at the same time, and see live scores.
- **Host**: The player who creates a realtime room and starts the match when everyone is ready.
- **Player identity**: Players enter a realtime room with a nickname only; account login is out of scope for the first version.
- **Live score**: A score where each round gives one point to the first player who submits the correct answer, immediately ending that round; wrong answers score zero and do not stop other players from answering.
- **Game mode**: The first version focuses on realtime rooms, while allowing a one-player room for testing.
